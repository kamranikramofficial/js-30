About This Project

The sticky navigation is implemented using JavaScript to detect scroll position and add/remove CSS classes. This creates a smooth transition effect when the navigation becomes fixed to the top of the viewport.

How It Works

  • Scroll Detection: JavaScript monitors the scroll position
  • Dynamic Classes: CSS classes are added/removed based on scroll
  • Smooth Transitions: CSS transitions create fluid animations
  • Responsive Design: Works perfectly on all screen sizes
script.js
window.addEventListener('scroll', () => {
  const navbar = document.getElementById('navbar');
  if (window.scrollY > 100) {
    navbar.classList.add('sticky');
  } else {
    navbar.classList.remove('sticky');
  }
});
