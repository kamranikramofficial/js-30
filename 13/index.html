Animate On Scroll

AOS is a library that allows you to animate elements as they scroll into view. This demo shows how to make AOS animations repeat every time an element enters the viewport.

Easy to Use

Just add data-aos attributes to your HTML elements

Customizable

Many animation types and options available

Performant

Uses CSS3 transitions for smooth animations

Animation Types

AOS provides many different animation types. Here are some examples:

fade-up

fade-down

fade-right

fade-left

zoom-in

zoom-out

flip-left

flip-right

Animation Options

AOS provides several options to customize animations:

Duration: 3000ms

data-aos-duration="3000"

Delay: 500ms

data-aos-delay="500"

Easing: ease-in-out-back

data-aos-easing="ease-in-out-back"

Anchor Placement

data-aos-anchor-placement="center-bottom"

How It Works

1

Include AOS Library

Add the AOS CSS and JavaScript files to your project.

<link href="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/aos@2.3.4/dist/aos.js"></script>
2

Initialize AOS

Initialize AOS in your JavaScript file.

AOS.init({
  // Global settings
  duration: 800,
  easing: 'ease-out',
  once: false // This makes animations repeat
});
3

Add Attributes to Elements

Add data-aos attributes to the elements you want to animate.

<div data-aos="fade-up">This will animate</div>
<div data-aos="fade-down" data-aos-delay="300">This will animate with delay</div>
4

Make Animations Repeat

To make animations repeat every time an element enters the viewport:

// Set once: false in AOS.init()
AOS.init({ once: false });

// Or use custom code to remove aos-animate class when elements leave viewport
document.addEventListener('scroll', function() {
  const elements = document.querySelectorAll('[data-aos]');
  elements.forEach(element => {
    if (!isElementInViewport(element)) {
      element.classList.remove('aos-animate');
    }
  });
});
